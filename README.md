# hola-mundo-qa-s2

Proyecto demo para la Sesion 2 del curso **Ruta de Transformacion QA Manual a QA Automation con IA**.

## Para correr los tests

```bash
npm install
npx playwright install chromium
npx playwright test
```

Credenciales de SauceDemo (publicas en la pagina): `standard_user` / `secret_sauce`.
